Ferramentas de Front-End

  • Bancos de imagens/vídeos/etc gratuios para serem usados

    • Algumas pessoas não sabem, mas nem todas imagens da internet/google tem permissão para ser utilizadas em sites ou aplicações, então o recomendavel é usar algum banco de imagens gratuito

  • Ferramenta de escolha para harmonização de cores

    • Site em que disponibiliza uma Ferramenta na qual podemos escolher uma ou mais cores e a própria API procura cores que tenha uma certa armonização com aquelas escolhidas, os códigos das cores também são disponibilizados, aperte a barra de espaço para que ele possa escolher

  • Ferramenta disponibilizada pelo Google para escolha e teste de fontes de letras
